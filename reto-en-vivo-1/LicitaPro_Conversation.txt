===============================================
INFORMACIÓN DEL PROYECTO ORIGINAL — CostaRica_Tenders
===============================================

Sistema de monitoreo automático de licitaciones gubernamentales de Costa Rica.
Compuesto por un web scrapper sobre SICOP y una API REST (Node.js) para consulta
y gestión de alertas por empresa.

Funcionalidades clave:
- Scraping automático de SICOP cada 2 horas
- Filtrado por tipos de licitación: [LN], [LI], [LA], [LY], [LE], [LD]
- Matching automático por productos y palabras clave
- Notificaciones por email
- API REST con autenticación JWT
- Endpoints para gestión de empresas, productos, palabras clave y licitaciones
